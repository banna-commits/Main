# Pattern: Blomsterbestilling (Mester Grønn)

## When
Knut wants to send flowers to Melissa (or others).

## Steps
1. Open browser (profile: openclaw) → mestergronn.no
2. Log in: knutgreiner@gmail.com / Uviktig1
3. Find product (default: 30 rosa roser, 804401)
4. Edit kort: "Elsker Deg" (tomt avsenderfelt)
5. Checkout → Privatadresse → Velg lagret mottaker "Melissa"
6. Velg dato + Ettermiddag/kveld leveringstid
7. Fortsett til avsender → Fortsett til betaling

## Gotchas
- Lagret mottaker "Melissa" under "Se lagrede mottakere"
- Totalpris ~750kr (560 roser + 15 kort + 175 frakt)
- Melissas bursdag: 23. februar — sett påminnelse i forkant!
